# 🌐 My Personal Website

This is a simple and responsive static website built using **React** and **Tailwind CSS**

---

## ✨ Features

- ⚛️ Built with React (Vite/CRA)
- 🎨 Styled using Tailwind CSS
- 📱 Fully responsive design
- 🔒 No backend, purely static
